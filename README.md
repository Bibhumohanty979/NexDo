NexDo – Your Smart Task Manager

A modern and responsive task management application built using HTML, CSS, and JavaScript. NexDo helps users organize daily tasks, manage priorities, track progress, and improve productivity through a clean and interactive interface.

----

Intern ID: CITS6090

Full Name: Bibhu Kalyan Mohanty

No. of Weeks: 4 Weeks

Project Name: NexDo – Your Smart Task Manager

Project Scope:
Developed a responsive task management web application that enables users to create, edit, organize, prioritize, search, and manage daily tasks efficiently. The application uses Local Storage for data persistence and provides a modern, user-friendly interface.

---

✨ Features

- 🔐 User Login & Logout
- 👤 Profile Icon
- 📊 Dashboard
- ➕ Add Tasks
- ✏️ Edit Tasks
- 🗑️ Delete Tasks
- ✅ Complete Tasks
- ↩️ Undo Completed Tasks
- 📌 Pin Important Tasks
- 🏷️ Task Categories
- 🚩 Priority Levels
- 📅 Due Date Management
- ⏰ Overdue Task Detection
- 🔍 Live Search
- 📋 All Tasks
- 📆 Today's Tasks
- ⭐ Important Tasks
- ✔️ Completed Tasks
- 🗑️ Trash & Restore
- 🌙 Dark Mode / ☀️ Light Mode
- 💾 Local Storage
- 📱 Responsive Design
- 🔔 Toast Notifications
- ⌨️ Keyboard Shortcuts
- 📤 Export Tasks (JSON)

---

🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
